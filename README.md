## How to Use (Now Fully Functional)
1. Open the live site
2. Click **Create New Auction** (or bid on the pre-loaded Grokzilla one)
3. Fill title, description, starting bid, duration, image URL
4. Watch live countdowns and place higher bids
5. When auction ends → Winner gets prominent **Claim Digital Asset** button
6. Claim generates unique unlock code + "delivered" confirmation (simulated email; real Gmail integration coming)
